# Kaffen Coffee & Restaurant Template

## Preview

Mở file `index.html` trong browser để xem preview.

## Sections

1. **Navigation** - Sticky navbar với logo và menu
2. **Hero** - Full-screen hero section với background image
3. **About** - Story section với stats và gallery grid
4. **Menu** - Menu items với tabs (Coffee, Food, Desserts, Drinks)
5. **Gallery** - Instagram-style photo grid
6. **Testimonials** - Customer reviews
7. **Reservation** - Booking form
8. **Contact** - Contact info với icons
9. **Footer** - Links và social media

## Design Features

- **Color Scheme:**
  - Primary: #C7A17A (Coffee/Gold)
  - Dark: #1A1A1A
  - Cream: #F9F5F0

- **Typography:**
  - Headings: Cormorant (Serif)
  - Body: Josefin Sans

- **Responsive:** Mobile-first design với Tailwind CSS

## How to Use

1. Mở `index.html` trong browser
2. Scroll xuống xem tất cả sections
3. Test responsive bằng cách resize browser
4. Khi OK, mình sẽ integrate vào Laravel
