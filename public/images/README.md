# Images Directory

## Gallery Images
Place restaurant gallery images in `gallery/` folder:
- `interior-1.jpg` - Restaurant interior
- `coffee-art.jpg` - Latte art
- `dining-area.jpg` - Dining space
- `ambiance.jpg` - Restaurant atmosphere
- `gourmet.jpg` - Gourmet dishes
- `coffee-beans.jpg` - Coffee selection
- `ingredients.jpg` - Fresh ingredients
- `dessert.jpg` - Desserts

**Recommended size**: 800x600px or 1200x800px
**Format**: JPG or PNG

## Menu Images
Already using placeholder URLs in seeder - can be updated to local paths if needed.

## Quick Setup
1. Download 8 restaurant/food images
2. Rename them to match the filenames above
3. Place in `public/images/gallery/`
4. Run: `php artisan migrate:fresh --seed`
